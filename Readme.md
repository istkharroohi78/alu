  <summary><b>🐳 DOCKER DEPLOYMENT</b></summary>

  ```bash
  git clone https://github.com/TEAM-ISTKHAR/ISTKHAR_MUSIC.git
  cd ISTKHAR_MUSIC
  nano .env
  docker build -t ISTKHAR-music .
  docker run -d --name ISTKHAR-music --env-file .env --restart unless-stopped ISTKHAR-music
  ```
  </details>

  <details>
  <summary><b>💻 VPS DEPLOYMENT</b></summary>

  ```bash
  sudo apt update && sudo apt upgrade -y
  sudo apt install git python3-pip ffmpeg -y
  git clone https://github.com/TEAM-ISTKHAR/ISTKHAR_MUSIC.git
  cd ISTKHAR_MUSIC
  pip3 install -r requirements.txt
  bash start
  ```
